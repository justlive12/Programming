В сервере были использованы следующие библиотеки:
import socket – сетевое взаимодействие через сокеты
from threading import Thread – использование многопоточности
import sqlite3 as sql3 – библиотека для работы с базами донных и языком SQL
import random – библиотека генератора случайных чисел
import string – библиотека работы со строками
import sys – библиотека работы с аргументами поданными серверу через командную строку

Данные библеотеки нужно импортировать до запуска программы
Для установки библиотек откройте командную строку и введите: pip install необходимая вам библиотека
